Detailed Description

An application examining the potential of the Random Ball Cover data structure for image segmentation.

This application was developed for testing the eligibility of RBC for image segmentation. Try changing the nr variable for configuring the number of representatives, and/or the a variable for adjusting the influence of the color intensity, and examine the effects on the image. By creating a graph with the representatives, and adjusting the edge weights, connected components could be created that would define areas of interest.

Note
The result is randomized. To assign representatives at regular intervals, try uncommenting the commented sections, and comment out the relevant lines (marked with // at the end).
